# SnapShop 📸🛍️

**Visual-Based Product Shopping Application**

SnapShop is an innovative image-based shopping web application that revolutionizes the e-commerce experience by enabling users to search for products simply by taking a picture. Powered by Django and integrated with LLAMA API, SnapShop transforms visual input into intelligent product recommendations from ebay.



## 🌟 Features

### Core Functionality
- **📷 Image Upload** - Intuitive drag-and-drop or file selection interface
- **🔍 AI-Powered Analysis** - Google Cloud Vision API extracts labels and object information
- **🎯 Smart Product Matching** - Intelligent matching of image labels with product database
- **⚙️ Product Management** - Complete CRUD operations for administrators
- **💻 Responsive UI** - Clean, modern interface with real-time feedback
- **📊 Results Display** - Comprehensive product information with prices and availability

### Bonus Features
- **👤 User Authentication** - Secure registration, login, and profile management
- **🔧 Advanced Filtering** - Filter by category, price range, and ratings
- **📜 Search History** - Track and revisit previous searches
- **📱 Mobile Responsive** - Optimized for all devices and screen sizes


## 🛠️ Technology Stack

### Backend
- **Django 4.0+** - Python web framework
-
- **Postgre SQL** - Database management system
- **LLAMA  API** - product display

### Frontend
- **HTML5** - Markup structure
- **CSS3** - Styling and animations
- **Bootstrap / Tailwind CSS** - Responsive design framework
- **JavaScript** - Interactive functionality

### Libraries & Tools
- **Pillow** - Image processing and handling
- **Requests** - HTTP library for API calls
- **python-dotenv** - Environment variable management
- **Git & GitHub** - Version control
